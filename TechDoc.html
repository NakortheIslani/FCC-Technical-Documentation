Flexbox is an extremely powerful layout and design tool in CSS. Before the Flexbox Layout module, there were four layout modes:

  • Block
  • Inline
  • Table
  • Positioned

The Flexbox Layout Module makes it easier to design flexible and responsive layouts without using float or postioning. Read on to learn more about the power of Flexbox!

Assumed knowledge

Before you go on to learn about Flexbox and its elements and properties, let's do a breif skills check. To understand Flexbox more easily, its recommended that you have at least a basic understanding of the following:

  • HTML structure and syntax
  • CSS structure and syntax
  • divs and containers
  • display properties
  • CSS selectors

Flex container (parent) properties

The flex container in the previous section showed an example of Flexbox. Now that we understand how to make all the HTML elements we need for a flex box, let's take a look at the CSS.

Example code

The container becomes flexible when you set the display property to flex.

.flex-container {
    display: flex;
}

Here is a chart listing all the Flexbox properties with breif descriptions.

PropertyDescription
flex-direction Specifies the direction of the flexible items inside a flex container.
flex-wrap Specifies whether the flex items should wrap or not, if there is not enough room for them on one flex line.
flex-flow A shorthand property for flex-direction and flex-wrap.
justify-content Horizontally aligns the flex items when the items do not use all available space on the main axis.
align-items Vertically aligns the flex items when the items do not use all available space on the cross-axis.
align-content Modifies the behavior of the flex-wrap property. It is similar to align-items,but instead of aligning flex items, it aligns flex lines.

Below, we are going to take a look at these properties and thier values.

The flex-direction property

The flex-direction property defines in which direction the container wants to stack the flex items.

A
B
C

Example code

The column value stacks the flex items vertically (from top to bottom).

.flex-container {
    display: flex;
    flex-direction: column;
}

Example code

The column-reverse value stacks the flex items vertically (but from bottom to top).

.flex-container {
    display: flex;
    flex-direction: column-reverse;
}

Example code

The row value stacks the flex items horizontally (from left to right).

.flex-container {
    display: flex;
    flex-direction: row;
}

Example code

The row-reverse value stacks the flex items horizontally (but from right to left).

.flex-container {
    display: flex;
    flex-direction: row-reverse;
}

The flex-wrap property

The flex-wrap property specifies whether the flex items should wrap or not. The examples below have 12 flex items, to better demonstrate the flex-wrap property. Example code

The wrap value specifies that the flex items will wrap if necessary.

.flex-container {
    display: flex;
    flex-wrap: wrap;

Example code

The nowrap value specifies that the flex items will not wrap (this is default).

.flex-container {
    display: flex;
    flex-wrap: nowrap;
}

Example code

The wrap-reverse value specifies that the flexible items will wrap if necessary, in reverse order.

.flex-container {
    display: flex;
    flex-wrap: wrap-reverse;
}

The flex-flow property

The flex-flow property is a shorthand property for setting both the flex-direction and flex-wrap properties. The direction value is entered first then the wrap value is set.

Example code

.flex-container {
    display: flex;
    flex-flow: row wrap;
}

The justify-content property

The justify-content property is used to align the flex items.

B

Example code

The center value aligns the flex items at the center of the container.

.flex-container {
    display: flex;
    justify-content: center;
}

Example code

The flex-start value aligns the flex items at the beginning of the container (this is default).

.flex-container {
    display: flex;
    justify-content: flex-start;
}

Example code

The flex-end value aligns the flex items at the end of the container.

.flex-container {
    display: flex;
    justify-content: flex-end;
}

Example code

The space-around value displays the flex items with space before, between, and after the lines.

.flex-container {
    display: flex;
    justify-content: space-around;
}

Example code

The space-between value displays the flex items with space between the lines.

.flex-container {
    display: flex;
    justify-content: space-between;
}

The flex-start value aligns the flex items at the top of the container.

Example code

The flex-end value aligns the flex items at the bottom of the container.

.flex-container {
    display: flex;
    height: 250px;
    align-items: flex-end;
}

Example code

The stretch value stretches the flex items to fill the container (this is default).

.flex-container {
    display: flex;
    height: 250px;
    align-items: stretch;
}

Example code

The baseline value aligns the flex items such as their text baselines aligns.

.flex-container {
